The standard markup language for creating web pages and web applications. HTML structures content on the web by defining elements like headings, paragraphs, links, images, and forms, forming the backbone of every website.

A stylesheet language used to describe the presentation of web pages, including layout, colors, fonts, and spacing. CSS enables responsive, visually appealing designs and enhances the user experience across different devices and screen sizes.

A powerful framework developed by Microsoft for building cross-platform, high-performance web applications, APIs, and microservices. It supports multiple programming languages and is widely used for enterprise-level applications.

A minimal and flexible Node.js web application framework that simplifies the development of web and mobile applications. Express.js provides a robust set of features for building APIs, handling HTTP requests, and managing middleware.

Git

jQuery

Visual Studio

An integrated development environment (IDE) by Microsoft that supports various programming languages and application types. Visual Studio provides powerful tools for debugging, testing, and version control, making it ideal for large-scale enterprise and cross-platform development.

Visual Studio Code

A lightweight, open-source code editor from Microsoft, known for its speed, flexibility, and extensive library of extensions. Visual Studio Code supports a wide range of programming languages and includes built-in Git support, making it popular for web development and cross-platform projects.
